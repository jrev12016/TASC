Steps to initiate:
    •	Open to directory with FinalProject inside
    •	Create environment with conda create -n [environment_name] python=3.7
    •	Activate environment with conda activate [environment_name]
    •	Install requirements with pip3 install -r requirements.txt
    •	Run python main.py and navigate to http://127.0.0.1:5000 in your browser
    •	Sign up or Login with a username & password 
    •	To access manager functions: username=manager & password=manager
    •	Enjoy!
